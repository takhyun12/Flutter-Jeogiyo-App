# jeogiyo

### Dart/Flutter를 공부하기 위한 습작

안드로이드 메터리얼 디자인을 통해 UI를 구현하였으며,
네이버 클라우드의 음성합성 서비스 "클로바"를 통해 TTS를 구현하였다.

식당에서 버튼을 누르면 이모를 호출하거나 물을 달라고 할 수 있다.
